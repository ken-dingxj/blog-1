# Tech stack
1. React v16
2. Webpack v4
3. Koa v2
4. Redux v3

# What to do
A todo list app, using server side rendering.
The app is universal(isomorphic), the components are available in `./client/src`.

# How to start
1. `cd client`
2. `npm install`
3. `npm run build`
4. `cd ../server`
5. `npm install`
6. `npm run server`
7. visit `localhost:667/todos`
